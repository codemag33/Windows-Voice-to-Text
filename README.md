# Windows Voice to Text

Простой голосовой ввод для Windows: удерживаете кнопку `2`, говорите, отпускаете — фраза распознаётся, копируется в буфер обмена и сохраняется в историю.

## Возможности

- Непрерывная работа в фоне.
- Режим записи по удержанию клавиши `2`.
- Автоматическое распознавание речи через Google Speech Recognition.
- Копирование результата в буфер обмена Windows.
- Сохранение истории распознанных фраз в JSON-файл.
- Работа в консоли Windows 10/11.

## Как это работает

- Пока клавиша `2` удерживается, программа записывает аудио с микрофона.
- Когда вы отпускаете клавишу, запись заканчивается.
- После этого фраза отправляется на распознавание.
- Результат выводится в консоль и копируется в буфер обмена.

## Требования

- Python 3.9+
- Windows 10/11
- Микрофон
- Интернет-соединение для `recognize_google()`

## Установка

```bash
pip install -r requirements.txt
```

Если `PyAudio` не ставится через `pip`, на Windows часто помогает установка подходящей wheel-версии для вашей версии Python. [web:62][web:24]

## Запуск

```bash
python gnu_win.py
```

## Управление

- Удерживайте `2` — запись.
- Отпустите `2` — распознавание.
- `Esc` — выход.

## Примечания

- Распознавание работает через Google Speech Recognition, поэтому нужен доступ в интернет. [web:1][web:10]
- Для микрофона библиотеке `SpeechRecognition` требуется `PyAudio`. [web:1][web:57]
- Если буфер обмена не работает, проверьте, что команда `clip.exe` доступна в Windows.
